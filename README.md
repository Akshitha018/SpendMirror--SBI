# SpendMirror-
AI personalised Financial Health Digest

SpendMirror is an AI-powered financial wellness assistant that analyses a user's spending patterns from bank transactions and generates a personalised weekly financial report. It automatically categorises expenses (food, travel, EMI, shopping, savings, etc.), compares spending habits with similar anonymous user groups, and provides simple AI-generated insights and suggestions to improve financial decisions.

The system converts complex transaction data into easy-to-understand messages like:

"Your food spending increased by 34% compared to similar users. Reducing food delivery twice a week can help you save ₹1200/month."

It delivers these insights through a dashboard and simulated WhatsApp/email notifications.
